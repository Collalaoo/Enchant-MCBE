# Enchant+ (Enchant-MCBE)

**Форк [Enchant-MCBE](https://github.com/DeathAruban/Enchant-MCBE)** — повністю переписаний аддон для Minecraft Bedrock Edition, який додає розширений стіл зачарування з підтримкою всіх ванільних зачарувань (включно з Lunge, Density, Breach, Wind Burst), українським інтерфейсом та зручною системою вибору.

## Особливості

- **Повна підтримка всіх ванільних зачарувань** — включаючи нові: *Lunge* (спис), *Density, Breach, Wind Burst* (булава)
- **Зручний UI через серверні форми** — інтуїтивний інтерфейс з випадними списками та повзунками (замість застарілого arucube UI)
- **Несумісні зачарування сіріють** — візуальне позначення (`§7§m`) неможливих комбінацій
- **Система конфліктів** — автоматичне блокування несумісних зачарувань (захист, зброя, інструменти тощо)
- **Зняття зачарувань** — видалення всіх або окремих зачарувань з поверненням досвіду
- **Перенесення зачарувань** — копіювання зачарувань між предметами (з перевіркою сумісності)
- **Український інтерфейс** — всі тексти, підказки та повідомлення українською
- **Оптимізований код** — чистий JavaScript з використанням `@minecraft/server@2.8.0` та `@minecraft/server-ui@2.8.0`

## Встановлення

1. Завантажте [`Enchant-MCBE.mcaddon`](https://github.com/Collalaoo/Enchant-MCBE/releases)
2. Відкрийте файл — Minecraft запуститься та імпортує аддон
3. Додайте обидва пакети (поведінки та ресурси) до свого світу
4. Створіть стіл зачарування: `Emerald + Obsidian + Enchanting Table`

## Використання

1. Розмістіть стіл зачарування
2. Натисніть правою кнопкою миші з порожньою рукою
3. Виберіть предмет з інвентаря
4. Оберіть дію:
   - **Зачарувати** — виберіть зачарування зі списку
   - **Зняти чари** — зніміть всі або окремі зачарування
   - **Перенести** — скопіюйте зачарування на інший предмет

## Технічні деталі

- **API:** `@minecraft/server@2.8.0`, `@minecraft/server-ui@2.8.0`
- **Мінімальна версія гри:** Minecraft 1.21.50+ (protocol 26.32)
- **Тип UI:** Серверні форми (ActionFormData / ModalFormData)
- **Вартість зачарування:** 3 рівні досвіду за рівень зачарування
- **Повернення при знятті:** 0.5 рівня за рівень
